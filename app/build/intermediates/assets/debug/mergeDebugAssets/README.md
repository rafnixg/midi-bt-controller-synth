# SoundFont Files

Place your SoundFont (.sf2 or .sf3) files here.

## Recommended Free SoundFonts

### Small Size (for mobile)
- **Salamander C5 Light** (~5MB) - Piano only
  https://musical-artifacts.com/artifacts/1009

- **OPL-3 FM 128M** (~10MB) - General MIDI, FM style
  https://musical-artifacts.com/artifacts/15

### Medium Size (Good Quality)
- **GeneralUser GS** (~30MB) - Full General MIDI
  https://schristiancollins.com/generaluser.php

- **SGM Soundfont** (~60MB) - High quality GM
  https://musical-artifacts.com/artifacts/855

### High Quality (Large)
- **FluidR3_GM** (~148MB) - Reference quality
  https://member.keymusician.com/Member/FluidR3_GM/

## Usage

1. Download a .sf2 file from one of the links above
2. Rename it or keep the original name
3. Place it in this `assets/` folder
4. The app will automatically load the first .sf2 file found

## Note

Larger SoundFonts provide better sound quality but:
- Increase APK size significantly
- Require more RAM on the device
- Take longer to load

For mobile apps, GeneralUser GS (~30MB) is a good balance.
